Implementation Summary

  • Main Page: Two separate buttons for creating bills
  • Dialog Title: Shows "Create New Bill (With GST)" or "Create New Bill (No GST)"
  • Info Section: Clear indication of selected GST option
  • Submit Button: Changes color and text based on GST option
  • Backend: Receives gst_enabled flag and processes accordingly

Expected Behavior

  1. User sees two buttons on main billing page
  2. Clicking "Create Bill (With GST)" opens dialog with GST enabled
  3. Clicking "Create Bill (No GST)" opens dialog with GST disabled
  4. Dialog shows clear indication of selected option
  5. Submit button reflects the chosen option
  6. Backend creates bill with appropriate GST settings
